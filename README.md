**AeroPlay**

A Frutiger Aero-inspired web music player that channels peak 2000s tech nostalgia (glossy glassmorphism, Aqua bubbles, and vibrant tropical green accents).


**Core Features**

Frutiger Aero UI: A deeply immersive, nostalgic skin built with glossy, glassmorphic web components and fluid animations.

Audio Feature Extraction: Real-time processing of uploaded audio tracks (`.mp3`, `.wav`, `.ogg`) to extract raw acoustic properties.

Algorithmic Mood Tracking: A custom multi-dimensional c classifier that tags tracks dynamically based on tempo and sonic energy.

AI Mood Detection: Upload an MP3 or WAV file, and our backend scans the rhythm and energy to instantly tag the track's mood.

**Tech Stack Simple Breakdown**
Frontend: React.js, Web Audio API, Canvas HTML5 (Deployed on Netlify)

Backend: Python, FastAPI, Librosa, Scikit-Learn (Deployed on Render)

Live demo available on : https://aeroplayer.netlify.app/
